  • Compline

    안정적으로,
    유용하게.

    COMPLINE은 관리적 취약점 진단을 위해 다양한 인증심사평가에 대한 컨설팅 환경을 제공합니다. 체계적으로 평가항목별 증적 파일 및 취약점 조치 이력을 관리하고 각 인증심사평가에 대응 가능한 보고서를 자동 생성할 수 있습니다.

    • dvr

      조치 현황 모니터링
      (통계 대시보드)

    • group

      취약점 조치
      워크 플로우 제공

    • finance

      CCE / CVE 취약점
      진단 결과 통합 관리

      관제 취약점(파일반 출입 / 장애처리 /
      침해사고 / 패턴관리 등) 관리기능
    • file_copy

      다양한 형태 별 보고서
      관리 기능 제공

      PPT, PDF, HWP, WORD, EXEL 등
      고객이 원하는 보고서 형태로 제공
    • shield_lock

      다양한 정보보호 인증/심사
      컨설팅 표준 플랫폼(V2.0)

    • lan

      취약점 진단 솔루션
      (CCE / CVE / CWE) API 연동

      ITSM 시스템, SSO, 인사시스템,
      메일시스템 등 다양한 시스템 연동

*전체적인 텍스트 수정이 필요한 섹션입니다.(컨펌된 텍스트 및 키워드 재삽입 예정)

Be best
